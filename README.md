# Overview

This project demonstrates the development of a Raspberry Pi-based robotic arm capable of performing pick-and-place operations through a custom Graphical User Interface (GUI).

The mechanical structure of the robotic arm was assembled using open-source 3D printable components, while the control system, GUI, and servo actuation were developed and integrated using Raspberry Pi.

The project focuses on robotic manipulation, embedded system integration, human-machine interaction, and real-time motion control.

# Working Principle

1. The user operates the robotic arm through a custom graphical user interface displayed on the 5-inch Raspberry Pi touchscreen.
2. Control commands are processed by the Raspberry Pi.
3. The Raspberry Pi generates control signals for each servo motor.
4. Servo motors position the robotic arm joints accordingly.
5. The robotic arm performs pick-and-place operations based on user commands.
